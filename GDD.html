Game Overview

Game Title: Re-Action

Game Description: The game is in the style of an infinite runner that will test the reaction time of the player.

Audience: Game is targeted towards all audiences.

Characters and Roles: John the Jumper(Main Character).

Environment: Varying environments; day, night, desert and city landscapes.

Theme: Reaction.


Gameplay and Mechanics

Objectives and Goals: Survive for as long as possible and gain points to set highscores.

Perspective: Sidescroller.

Controls: Standard WASD movement layout, spacebar to jump, shift to slide.

Reference Points and Originality: Reference taken from other infinite runners and aspects of sidescrolling platformers.


Technical Requirements

Platform: Website game.

Development Environment: Unity3D.

System Requirements: 128MB RAM minimum.

Resourcing and Capability: All of the resources within the game and all images on the site are created by the team members.


Visuals, Artwork and Graphics

Style: Pixelated graphics

Process: Adobe Photoshop and Illustrator.
